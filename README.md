тестовое задание

Откройте проект в VS Code.
Установите расширение **Live Server** (если еще не установлено).
Откройте `index.html` и нажмите **Open with Live Server**.
Страница откроется автоматически в браузере.
Основной результат: верстка + Vue 3 интерактивность в `index.html`.
Стили для просмотра берутся из `public/assets/css/main.css`.
JS-логика берется из `public/assets/js/main.js`.
SCSS-исходники находятся в `resources/scss/` (как плюс к ТЗ).
